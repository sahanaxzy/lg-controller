# Liquid Galaxy Controller

A Flutter application to control a Liquid Galaxy. 

## Features
* Connects to Liquid Galaxy via SSH
* Explores world monuments
* Fly-To animations to specific Google Earth coordinates
* Displays monument info panels and the Liquid Galaxy logo on the slave screens using KML
* Smooth orbit around monuments

## Usage
1. Connect an Android device to the Liquid Galaxy Wi-Fi network.
2. Enter the Master node IP address,SSH port, Username and password in Settings.
3. Tap on a monument to fly there and start exploring!
